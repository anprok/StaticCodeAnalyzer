Nested list

Containers in Python don't store values inside, instead they store references to other objects. This allows us to do crazy things, for example, put a list inside itself.

empty_l = []
empty_l.append(empty_l)

print(empty_l)    # [[...]]

When Python sees that a list lies inside itself, it replaces the list with [...] in output.

What will be the output of print(empty_l.copy())?

Just launch the code in an interpreter.
